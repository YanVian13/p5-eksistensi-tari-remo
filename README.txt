README - Cara pakai
Folder ini berisi website satu-halaman (HTML/CSS/JS) untuk Proyek Tari Remo. Tema utama telah diubah ke dark mode dengan palet merah; terdapat tambahan elemen visual yang terinspirasi Tari Remo.
- Buka index.html di browser untuk melihat tampilan.
- Jika ingin mengganti font atau warna, edit styles.css.
- Untuk menambahkan audio, letakkan file MP3 di folder /audio dan beri nama sample-remo.mp3
- Untuk deploy: cukup upload seluruh folder ke Netlify, Vercel (static), atau hosting statis apa pun.
